This project is practice project to learn bootstrap from scratch.
Build html pages with boostrap.
Boostrap version :  Bootstrap v4.1.3. You can download the compiled version from https://getbootstrap.com/
We use http://lorempixel.com/ for image sources and placeholder.com
There are many ways you can set up boostrap in your project.

1. Using CDN
    You can refer the CDN Url in your html pages to use bootstrap
2. Download compiled version directly 
    You can download compiled version of bootstrap from the website and refer it in your project html pages
3. Use npm package manager
    Use Node package manager to download the files into your project
